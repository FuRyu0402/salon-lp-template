# Responsive Hair Salon LP Template

Tailwind CSSを使用した、シンプルでモダンな美容室・サロン向けのランディングページ（LP）テンプレートです。

## 特徴
- **完全レスポンシブ対応:** スマートフォン、タブレット、PCそれぞれの画面サイズに最適化し、実用的なハンバーガーメニューを実装しています。
- **CSSアニメーションによるファーストビュー:** JavaScriptに依存せず、CSSのみで軽量に動作するフェードイン・アップおよびスライドショーを搭載しています。
- **高メンテナンス性（非同期通信の活用）:** お知らせ（INFORMATION）部分は、外部HTMLファイルを非同期（Fetch API）で読み込む設計にしています。全体のコードを書き換えることなく、特定ファイルのみの更新で運用が可能です。
- **パフォーマンス最適化:** `<picture>`タグを用い、次世代画像形式（WebP）への対応（非対応ブラウザへのフォールバック含む）を行っています。

## 構成ファイル
- `index.html` - メインのHTML構造
- `style.css` - 独自のアニメーションおよびフォントスタイル
- `main.js` - ナビゲーション制御および非同期通信処理

## 使用技術
- HTML5 / CSS3
- Tailwind CSS (CDN)
- Vanilla JavaScript (ES6)
